Leonardo integration

https://leonardocolor.io

This app demonstrates how Leonardo can be used to create an adaptive color palette. The user can control the theme's lightnerss, contrast, and saturation using the sliders at the top of the page.

The sliders pass inputs to the Theme setters, which return new colors from Leonardo. Colors are implemented as CSS properties so that Leonardo can dymanically update their values.
