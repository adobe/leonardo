Palette auditor

One hex per line. Rows = text color, columns = background. AA needs ≥ 4.5:1, AAA needs ≥ 7:1 (WCAG 2.1 normal text).

Each row uses the color as a foreground; each column uses it as a background. Invalid or empty lines are ignored.

WCAG 2.1 thresholds

AA
≥ 4.5:1 — normal text
AAA
≥ 7:1 — normal text (enhanced)

Anything lower fails that level for normal text.
